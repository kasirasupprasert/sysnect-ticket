v1.7.0 - Unified Asset Dashboard & Lazy Load 🖥️📦
  • ปรับโฉมและโครงสร้างหน้าเว็บใหม่ทั้งหมด (Unified Dashboard) ให้มีความเรียบง่าย คลีน และรวมทุกฟังก์ชันไว้ในหน้าเดียว
  • เพิ่มหน้ารายการทรัพย์สิน (Asset) สำหรับติดตามข้อมูลอุปกรณ์ ตรวจสอบวันหมดอายุ และสถานะการรับประกันต่างๆ
  • อัปเกรดระบบการดึงข้อมูล (Data Fetching) เป็นแบบ Lazy Loading โดยระบบจะโหลดข้อมูลก็ต่อเมื่อผู้ใช้กดสลับไปยังหน้าหรือแถบนั้นๆ ช่วยให้เว็บตอบสนองไวขึ้นและลดการใช้ทรัพยากรเซิร์ฟเวอร์
v1.6.0 - Cleaner Navbar & Smart Bell 🔔📊
  • ออกแบบแถบเมนูด้านบนใหม่ทั้งหมด แยกฟิลเตอร์ (เวลา/สถานะ/ความสำคัญ/โครงการ) ออกมาเป็นแถบตัวกรองของตัวเอง ลดความแออัด อ่านง่ายและกดง่ายขึ้น
  • ย้ายสวิตช์ "เปิดระบบแจ้งเตือน" ออกจากหน้าคุณสมบัติระบบ มาเป็นปุ่มกระดิ่ง 🔔 บนแถบเมนู กดครั้งเดียวเปิด/ปิดได้ทันที พร้อมจุดไฟแสดงสถานะเปิดอยู่
  • ปุ่มสลับธีม (Dark/Light) ย้ายออกมาจากเมนู ... ให้เป็นไอคอนลัดกดง่ายอยู่บนแถบเมนูโดยตรง
  • เพิ่มปุ่มสลับมุมมองกราฟโดนัทภาพรวม ให้ดูสัดส่วนงานแยกตาม "ผู้รับผิดชอบ" ได้ นอกเหนือจากมุมมองตามสถานะเดิม
v1.5.0 - Smart Notify Banner 🔔
  • ลบระบบ Auto-refresh อัตโนมัติออก เพื่อไม่ให้รบกวนการทำงาน
  • เพิ่มระบบ Background Polling เช็คข้อมูลใหม่เบื้องหลังทุก 1 นาที โดยไม่ reload หน้า
  • เมื่อพบ Ticket ใหม่: แสดง Notify Popup บนสุดหน้าจอพร้อมเสียงแจ้งเตือน และให้ผู้ใช้กดรีเฟสเองเพื่อดูข้อมูล
v1.4.9 - Job Duration Tracker ⏱️
  • เพิ่มระบบแสดงอายุงาน (Duration) บนการ์ด Ticket ทุกใบ แสดงเป็น ปี เดือน วัน
  • งานที่ยังเปิดอยู่: นับระยะเวลาจากวันเปิดถึงปัจจุบัน พร้อมสีบ่งบอกความนาน (เขียว/เหลือง/แดง)
  • งานที่ปิดแล้ว (Closed/Solved): แสดงระยะเวลาทั้งหมดตั้งแต่เปิดจนถึงปิด พร้อม Export ลง CSV
v1.4.8 - Changelog Archive 📚
  • จัดระเบียบหน้าประวัติการอัปเดต โดยโชว์เฉพาะรายการล่าสุด และมีปุ่มให้กดกางเพื่อดูประวัติเก่าๆ ทั้งหมด
v1.4.7 - UI Fix for Date Picker 🗓️
  • ปรับปรุงดีไซน์ช่องระบุช่วงเวลา (Custom Date) ให้ตัวหนังสือคมชัดขึ้น ไม่แตกเบลอ เพิ่มความสวยงามและกดง่ายขึ้น
v1.4.6 - Custom Date Range 📅
  • เพิ่มฟีเจอร์ "ระบุช่วงเวลา (Custom Date)" ให้ผู้ใช้สามารถดึงข้อมูลแบบเจาะจงวันที่เริ่มต้นและสิ้นสุดได้เองตามต้องการ
v1.4.5 - Status Sync Updates & Pop-up 🔄
  • เพิ่มระบบ Smart Pop-up เด้งแจ้งเตือนการอัปเดตใหม่ๆ (แสดงผลแค่ครั้งเดียว)
  • ปรับปรุงระบบ Priority: หากสถานะงานเป็น Closed จะถูกจัดความเร่งด่วนเป็น "ปิดแล้ว (Closed)" ให้อัตโนมัติ
v1.4.4 - Priority Filtering 🔥
  • เพิ่มเมนูตัวกรอง (Dropdown Filter) เพื่อดูงานตามระดับความเร่งด่วน (Priority)
v1.4.3 - Refined Updates ✨
  • ปรับปรุงระบบ Time Span ให้คำนวณและแสดงผลเป็น "ปี" ได้หากข้อมูลเกิน 365 วัน
  • เพิ่ม Filter สำหรับค้นหาข้อมูล "ปีนี้" และ "ปีที่แล้ว"
  • แก้ไขการแสดงผลจุดสี (Legend Dot) บนเมนูฝั่งขวาให้สวยงามและจัดเรียงถูกต้อง
v1.4.2 - Time Span Logic 🕒
  • เพิ่มป้ายแสดง "ข้อมูลย้อนหลัง" (Time Span) เพื่อคำนวณและแสดงจำนวนเดือน/วันที่ดึงข้อมูลมาอัตโนมัติ
  • แก้ไข CSS จุดสีในรายการ Status ฝั่งขวาให้แสดงผลถูกต้อง
v1.4.1 - Design Upgrade ✨
  • ปรับปรุง Layout หน้าต่าง Sidebar จัดกลุ่มป้ายกำกับ Meta ให้สวยงามขึ้น ป้องกันปัญหาข้อความยาวทะลุกรอบ
  • ปรับดีไซน์ป้าย Priority ให้เป็นแบบ Soft UI สีพาสเทลขอบมน สบายตา
v1.4.0 - Priority System 🔥
  • เพิ่มระบบดึงข้อมูลระดับความสำคัญ (Priority) จาก GLPI และเพิ่ม Badge แจ้งเตือนความด่วนบนการ์ด
v1.3.0 - UI Enhancements 🎨
  • ปรับปรุงหน้า UI (Clean Interface) รวบเมนูต่างๆ ไว้ใน Dropdown เดียว
v1.2.0 - Mode Switching 🌙
  • เพิ่มระบบ Dark Mode / Light Mode พร้อมจำค่าเริ่มต้น
  • เพิ่มระบบแจ้งเตือนเชิงรุก (Active Alerts) แบบเสียงและ Desktop Pop-up
  • อัปเกรดระบบ Export เป็นไฟล์ CSV
v1.0.0 - Initial Release 🚀
  • เริ่มการใช้งานระบบ Ticket Management Overview อย่างเป็นทางการ
